 # ProgramAndWeb

 A simple single-page website showcasing web development services. This repository contains the project's static site files (HTML, CSS, and JavaScript).

 ## Project Structure

 - `index.html` — Main website file (landing page).
 - `README.md` — This file.

 ## Features

 - Responsive layout with a hero section, services, and contact area.
 - Lightweight, dependency-free HTML/CSS/JS (uses Font Awesome CDN for icons).
 - Smooth scrolling and simple contact mailto link.

 ## Run locally

 1. Open the site directly: open `index.html` in your browser.

 2. Or serve with a simple HTTP server (recommended for testing relative paths):

 ```bash
 python3 -m http.server 8000
 # then open http://localhost:8000 in your browser
 ```

 ## Customize

 - Edit `index.html` to update text, sections, email address, or assets.
 - Replace the background image URL and colors in the `<style>` block to match your branding.
